# Design Tool Evaluation

| Criterion | Tool A: diagrams.net | Tool B: PlantUML Web |
|---------|----------------------|----------------------|
| Learning curve | Very easy, drag-and-drop interface | Requires learning UML syntax, fast after basics |
| Version control friendliness | .drawio files can be diffed as XML; PNG used for review | .puml is text-based and diff-friendly; PNG used for review |
| Validation support | Manual visual checks only | Syntax validation and basic linting via editor |
| Collaboration | Works well with Google Drive / OneDrive | PR-friendly; easy code reviews |
| Export formats | PNG, SVG, PDF | PNG (regenerated from source) |
| Cost | Free | Free |
